Soft Robotic Glove

Description

  • Objective:
    • Create a prototype for a proof-of-concept medical device utilizing soft robotics
  • Result:
    • Developed a wearable glove that will assist user with finger flexion controlled by an electromyography(emg) sensor. The user would wear this product and attach an EMG to their forearm. When the user flexed their fingers, the EMG would pick up the electric signals and activate the air pump, triggering the soft actuators, that would replicate a flexing motion.
  • Role:
    • Researched and optimized the design of a soft actuator component
    • Created mold for soft actuator by 3D printing component
    • Urethane casted soft actuator using DragonSkin
    • Programmed and tweaked EMG to control air pump based on user muscle activation
