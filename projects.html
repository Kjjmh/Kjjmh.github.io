Soft Robotic Glove

Previous Next

Description

  • Objective:
    • Create a prototype for a proof-of-concept medical device utilizing soft robotics
  • Result:
    • Developed a wearable glove that will assist user with finger flexion controlled by an electromyography(emg) sensor. The user would wear this product and attach an EMG to their foremrm. When the user flexed their fingers, the EMG would pick up the electric signals and activate the air pump, triggering the soft actuators, that would replicate a flexing motion.
  • Role:
    • Researched and optimized the design of soft actuator component
    • 3D printed mold shell for soft actuator
    • Urethane casted soft actuator
    • Programmed EMG to control air pump
